Sistema de Reservas de Pistas Deportivas

Esta aplicación permite gestionar de manera completa la reserva de pistas deportivas, desde el registro de usuarios hasta el cálculo de ingresos y la gestión de cancelaciones. Está diseñada para ser utilizada tanto por administradores como por usuarios normales, con control de accesos y validaciones robustas.

Funcionalidades principales

Gestión de usuarios: Registro, edición y eliminación de usuarios. Control de roles (ADMIN/USER).

Gestión de pistas: Creación, edición y eliminación de pistas (solo administradores). Cada pista tiene nombre, tipo y ubicación.

Reservas: Los usuarios pueden reservar pistas disponibles en franjas horarias específicas. Se valida la disponibilidad y se calcula el precio según horarios pico/valle.

Visualización: Listado de pistas disponibles según fecha y hora. Historial de reservas de cada usuario.

Cancelación de reservas: Política de cancelación flexible:

24h antes: 100% reembolso

12–24h antes: 50%

<12h antes: 0%

Ingresos: Cálculo de ingresos por pista en un período
